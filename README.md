# API-Testing-Example

REST API testing starter suite.

Overview
--------

This repository provides an example how to test REST API's with Copado Robotic Testing.

Examples of GET operation using an open API (OpenLibrary). 3rd party library "RequestsLibrary" is used here for making API calls.

Additional information
----------------------

Directory structure:

* Tests are under "tests" folder
* Supporting files are under "resources" folder
